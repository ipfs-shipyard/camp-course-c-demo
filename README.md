# IPFS Camp 2019 Course C Materials

## To run locally

(Must have a recent [Node.js](https://nodejs.org/en/) installed)

```
git clone https://github.com/ipfs-shipyard/camp-course-c-demo.git

cd camp-course-c-demo

npm start
```
